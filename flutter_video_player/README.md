# Flutter Video Player

A Flutter application playing online video with custom player controller.

## Getting Started

使用 [video_player](https://pub.dev/packages/video_player) plugin，製作一個影片播放器，並且可以播放、暫停、調控音量等，自行客製影片 controller。
